# CineVerse 기능 명세서

---

## 서비스 개요

영화 캐릭터와 대화하고 영화를 추천받는 AI 서비스.  
50개 영화 캐릭터와 1:1 대화, 그룹 채팅, 영화 추천을 지원합니다.

---

## 아키텍처

```
프론트엔드
    ↓ (HTTP)
백엔드 서버 (JWT 인증, 히스토리 관리, DB 저장)
    ↓ (HTTP)
AI API 서버 (210.109.15.251:80)  ← FastAPI + uvicorn
    ↓
┌─────────────────────────────────┐
│  인텐트 분류기 (pipeline/intent) │
└─────────────────────────────────┘
    ↓                  ↓
캐릭터 파이프라인    영화 추천 파이프라인
    ↓                  ↓
llama-server:8081  Milvus:19530 (벡터 검색)
(Gemma-4 12B Q4_K_M)
```

## 핵심 기능

### 1. 인텐트 자동 분류
- 사용자 메시지를 분석해 `character_chat` / `movie_recommend` 자동 판별
- `/chat/auto` 엔드포인트에서 처리
- 캐릭터 선택 전 영화 추천 질문도 처리 가능 (`character` 생략 가능)

---

### 2. 1:1 캐릭터 대화
- 50개 캐릭터 중 하나와 대화
- RAG(Retrieval-Augmented Generation)로 캐릭터 관련 기억/맥락 주입
- 멀티턴 대화 지원 (`history` 파라미터)
- 캐릭터별 말투·성격·세계관 반영

**엔드포인트**: `POST /chat` 또는 `POST /chat/auto`

---

### 3. 그룹 채팅 (단순)
- 2~5명의 캐릭터가 동시에 사용자 메시지에 답변
- 앞 캐릭터의 답변을 히스토리로 참고하며 순차 답변
- 캐릭터 간 중복 표현 방지

**엔드포인트**: `POST /chat/group`

---

### 4. 그룹 채팅 (반응형, 2라운드)
실제 단체 대화방처럼 캐릭터들이 서로 반응하는 방식

**Round 1** — 사용자 메시지에 각 캐릭터 순차 답변  
**Round 2** — 1라운드 전체 대화를 보고 캐릭터들이 자율 반응
- 다른 캐릭터를 직접 지목해 동의·반박 가능
- 할 말 없으면 침묵 (응답 목록에서 제외)
- 자신이 이미 한 말 반복 금지

**엔드포인트**: `POST /chat/group/rounds`

---

### 5. 영화 추천
- 사용자 요청에 맞는 영화 추천
- 캐릭터 지정 시 해당 캐릭터 스타일로 추천
- 필터 지원: 장르, 배우, 감독, 언어, 개봉연도, 최소 평점

**엔드포인트**: `POST /recommend`

## 캐릭터 시스템

### 프로파일 구조
각 캐릭터는 다음 정보를 보유합니다.

| 항목 | 설명 |
|------|------|
| identity | 캐릭터 핵심 정체성 |
| personality | 성격 특성 |
| speech_style | 말투·어조 |
| thinking_style | 사고방식 |
| signature_elements | 고유 관점 |
| interaction_style | 사용자/다른 캐릭터 대응 방식 |
| multi_chat_behavior | 그룹 채팅 전용 행동 규칙 |
| examples | 말투 예시 |
| avoid | 반드시 피할 표현 |

### 프롬프트 모드
| 모드 | 용도 | 특징 |
|------|------|------|
| compact | 서비스 (1:1, 그룹) | ~600토큰, 핵심 필드만 |
| full | 테스트·튜닝 | 전체 필드 |

## RAG (검색 증강 생성)

- **DB**: Milvus (벡터 데이터베이스)
- **임베더**: BGE-M3 모델
- 캐릭터 관련 원작 기억·대사를 검색해 프롬프트에 주입
- 1:1 대화: top-3 검색, 그룹 채팅: top-2 검색
- 실패 시 무시하고 진행 (서비스 중단 없음)

## LLM

| 항목 | 내용 |
|------|------|
| 모델 | Gemma-4 12B (fine-tuned) |
| 포맷 | GGUF Q4_K_M |
| 서버 | llama-server (포트 8081) |
| GPU | Tesla T4 15GB |
| 컨텍스트 | 4096 토큰 |
| 응답 토큰 | 1:1 대화 512, 그룹 1라운드 512, 그룹 2라운드 256 |

---

## 제약 사항

| 항목 | 제한 |
|------|------|
| 그룹 채팅 캐릭터 수 | 2~5명 |
| 응답 길이 | 1~3문장 |
| 타임아웃 | 30초 이상 권장 |
| 동시 요청 | 1 (llama-server np=1) |

---

## 주요 금지 사항 (캐릭터 공통)

- 원작 사건·무용담 회상 금지
- "힘내", "포기하지 마" 등 범용 상담 문구 금지
- 실제 폭력·범죄 직접 권유 금지
- 마크다운 형식 출력 금지 (순수 대사만)
- 다른 캐릭터 대사를 대신 생성 금지 (그룹 채팅)
